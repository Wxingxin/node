

# 🟦 一、**Web 服务器相关（Express、Koa、Fastify 等）**

### ✔ Express 常用包

| 包名                   | 作用                 |
| -------------------- | ------------------ |
| **express**          | Node 最常用的 Web 框架   |
| **cors**             | 跨域处理               |
| **body-parser**  （已内置） | 解析 JSON、表单数据       |
| **cookie-parser**    | 解析 cookie          |
| **express-session**  | Session 会话管理       |
| **morgan**           | HTTP 请求日志工具        |
| **compression**      | Gzip 压缩响应数据        |
| **helmet**           | 提高安全性（禁用危险 header） |
